# 🍴 Smart Recipe Finder

## 📌 Description
React-based recipe finder application that integrates external APIs to provide real-time search, smart suggestions, and detailed recipe information.

## 🚀 Tech Stack
- React.js
- JavaScript
- React Router DOM
- Context API
- Fetch API
- CSS
- AOS (Animate On Scroll)

## ✨ Features
- Search recipes in real time using Spoonacular API
- Display recipe details including ingredients and nutrition
- Video integration for cooking tutorials
- Add-to-cart functionality using Context API
- Responsive UI for all screen sizes
- Smooth animations using AOS
- Toast notifications for user interactions

## 🔗 Integration
- Integrated with Spoonacular API for recipe and nutrition data
- Fetch API used for real-time data retrieval
